Character groups — a best-of-64 grid

A live snapshot of the growing laion/moss-character-voices-bestof64 dataset, generated with MOSS-TTS-Local 4.55B, zero-shot and no reference audio. Each group pairs a character's fixed, optimized champion instruction with a different spoken text (a Gemma-authored topic line); the model draws 64 takes per group, and every take is scored and reward-ranked within its own group.

How to read this grid

Every character has one fixed champion instruction — the voice prompt that won that character's earlier best-of-64 tournament. We then hold that instruction constant and feed the model many different spoken texts, each a short line generated around a topic word. For each (instruction × text) pair we sample 64 takes and score them with a per-character reward:

reward = ( w_blend·blend + w_genu·genu + w_vn·char_vn + w_emo·char_emo ) × WER-factor, then min-max normalized within the 64-take group.

blend = timbre/style match · genu = naturalness/genuineness · char = VoiceNet character-dimension fit · char_emo = EmoNet emotion fit · invWER = word-accuracy factor. Rank 1 is the best take in the group.

Each group card below shows the topic and language, the spoken text, the (collapsible) fixed champion instruction, and the group's top-3 takes as audio players: ① Champion, ② Runner-up, ③ Third — the best three of 64 — each tagged with its reward and the component scores.

Snapshot notice. The dataset is still generating — it currently holds ~5,900+ groups across 59 shards. Six characters have crossed the finish line so far and are shown here in full: the dragon, evil ghost, fairy and goblin creatures, plus the two ASMR voices. The remaining characters — mouse, ork, pain scream, ranting, sad man, sad woman and zombie — will be added here as generation reaches them. Follow progress at the dataset page.
